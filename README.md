# chat-challenge

## Project setup
First, you need is configure your API URL. Go to the follow path: src/Config/urls.ts.

```bash
 export default {
     ApiURL: "https://localhost:44368/api",
     MainApiURL: "https://localhost:44368/"
}
```
Then install the dependencies
```
npm install
```
